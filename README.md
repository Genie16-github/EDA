# :rocket:EDA
**날씨에 따른 소비 패턴 변화를 기반으로 한 상품 추천**
<br/><br/>
한국은 다양한 날씨가 존재하고 그에 따라 다른 소비 패턴을 확인할 수 있다. 날씨에 따른 소비 패턴을 분석하고, <br/>개인에게 제품을 추천해주는 개인화 마케팅 전략을 세웠다.
